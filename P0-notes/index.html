How The Web Works :

when you enter something like google.com the request goes to one of many special computers on the internet known as domain servers (DNS).All these requests are routed through various routers and switches.The domain name servers keep tables of machine names and their IP addresses,so when you type in google.com it get translated into a number, which identifies the computers that serve the google website to you. When you want to view any page on the Web, you must initiate the activity by requesting a page using your browser. The browser asks a domain name server to translate the domain name you requested into an IP address. The browser then sends a request to that server for the page you want, using a standard called Hypertext Transfer Protocol or HTTP. HTML :

HTML stands for HYPER TEXT MARKUP LANGUAGE. The web consists of majiority of HTML documents. HTML documents contain Text Elements which determine "What User See's" as well as Markup Elements which refers to "How It Looks".HTML is the Language of Web Development. Usually HTML documents are stored on servers, which provide the necessary documents to the users whenver a user acceses a webpage. The data is sent to the user's System which is furthur interpretted by the Web Browser in use, this data is then displayed to the user.
