Webpages, Documents, and Structure

The Basics of the Web and HTML

How the Web Works

Hyper Text Markup Language is the language of the internet. HTML is interpreted by a computer's browser. These documents reside on servers. When a user accesses a webpage their computer sends an HTTP request to the server. The server finds the document and sends it across the internet where it is interpreted by the user's browser. The web is a collection of HTML documents. There are approximately 30 billion pages on the World Wide Web.



HTML Markup

  • Text Content- "What you see"
  • Markup- "What it looks like" references to other documents e.g. images and video and links to other pages.
  • Element-An element is comprised of an opening tag, content, and a closing tag.

HTML Attributes

  • Tag Attribute< tag attribute="value" > contents</tag>
  • Anchor Tag <a>- creates a link
  • Creating a link <a href="www.reddit.com" >derp</a> derp
  • image tag- <img><img src="url of image" alt="text">

White Space

  • HTML renders all white space as a single space

Inline vs. Block

  • <span></span>- inline
  • <div></div>- block

Creating a Structured Document with HTML

Understanding CSS

CSS is code written to control the "style" of HTML elements.

h1 {
background-color : black;
color: white;
}

h1 is a selector. This tells the browser what to apply the rules too. The rules are between the {} braces.background-color : black is a rule. background-color is an attribute and black is the value of that attribute. To add style to specific h1 or other element use class selectors. Class selectors are written in the format .class-name

Adding CSS to a webpage

CSS can be added in three ways.

  1. Write CSS in the <head> of your HTML. Best for small projects and testing.
  2. Link your HTML to a separate CSS file. Most organized method and best for larger projects.
    1. Write all of your structural HTML in one file (let's call it main.html).
    2. In a separate file (let's call it main.css), put all of your CSS code.
    3. Add a <head> element to the top of your HTML.
    4. Add a <link> tag inside the <head> element. Since <link> is a "void tag" you don't need to add a closing </link>.
    5. Add the following attributes to your <link rel="stylesheet" type="text/css" href="main.css">
  3. Write your style inline with your HTML- This results in a lot of repeated code.

Input-->Function-->Output

Functions or "Procedures"

A function is something that takes input, does something to that input, and then produces output. The value of a function is that it allows programmers to do reuse code, because once it is defined you don't have to define it again. To create a function in Python you use the following method.

def square(x): #use keyword def, followed by name, and then parentheses around the parameters
answer = x * x
return answer
# to call the function, write the name and give it values
print square(4)
>>>16

The return keyword is vital to the function because it tells Python what it should produce as an output.

The built-in method index gives the position of the first match within a list of a particular element.

def find_element(list,value):
 for e in list:
  if e == value:
   return list.index(e)
 return -1

Python Standard Library

The Python Standard Library contains built-in functions that do not require the use of the import command. Python also contains modules with built in functions that must be "imported" in order to be used. Python Standard Library A module is a file that contains functions or executable and can be imported into other modules or the main module.

Abstraction

Abstraction in programming is a way of managing complexity. The specific way a function works are hidden behind documentation and the details are hidden from the programmer. This allows the programmer to focus on the program we want to write. Wikipedia Article on Abstraction

Use Classes

Within the Python Standard Library are files not only with functions, like in the previous section, there are also files with Classes. Classes are neatly packaged boxes that put things together very well. The following is an example of the Python Module Turtle

brad = turtle.Turtle() # Creates an instance of the class "Turtle" named brad
brad.shape("turtle") # Shape of a turtle
brad.color("yellow") # Color will be yellow
brad.speed(2) # Sets the speed
 Class- Think of it as a blueprint of a building.
Object - Instances or examples of that blueprint.

When calling webbrowser.open() we are simply calling a function. When calling turtle.Turtle() it actually calls the init function within turtle and creates space within memory that did not exist before.
